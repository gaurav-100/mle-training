# Docker installation

1. Docker used in root
    `sudo -i`

2. In WSL to start docker,
    `sudo service docker start`

3. `docker run -dit debian`
    - 'd' denotes **detach**, means it can run in background
    - 'i' denotes **interactive**
    - 't' denotes allocates sudo terminal
